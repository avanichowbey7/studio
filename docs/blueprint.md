# **App Name**: Swasthya Mitra

## Core Features:

- Language Selection: Allow users to select their preferred language (English, Hindi, Marathi).
- Appointment Booking: Enable users to book appointments with available doctors.
- Doctor Profiles: Display information about doctors, including their specialization and availability.
- AI-Powered Intelligent Suggestion Tool for Doctor: The system leverages LLMs to recommend the most suitable doctor or department based on the user's input description of their symptoms and their medical history to provide personalized suggestions.
- Help Section: Provide a help section with frequently asked questions and contact information.

## Style Guidelines:

- Primary color: Calming light blue (#ADD8E6), reflecting trustworthiness and serenity associated with healthcare.
- Background color: Very light desaturated blue (#F0F8FF) to create a soft, clean backdrop.
- Accent color: Muted green (#90EE90), complementing the primary color and resonating with health and growth.
- Body and headline font: 'PT Sans', a humanist sans-serif for readability and a modern feel.
- Use clean, vector-based icons relevant to healthcare, appointments, and navigation. Icons should be accessible and easily recognizable.
- 16:9 aspect ratio, left-side vertical sidebar with language selector and navigation. Main content area on the right with soft gradient background and center-aligned text. Prioritize accessibility and elderly-friendliness.
- Subtle animations for button interactions and transitions between pages to enhance user experience without overwhelming.
